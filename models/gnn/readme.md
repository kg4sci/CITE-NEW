## GNN

#### Environment requirements

- Python  >= 3.6
- PyTorch >= 2.3.0

- DGL >= 2.2.1

```
pip install -r requirements.txt
```

#### Datasets

download [here](https://huggingface.co/datasets/kg4sci/CITE/blob/main/chemistry.pt) and put it on ../../dataset

#### Run

```
chmod +x run.sh
./run.sh
```
